# Pinata IPFS File Upload JSON Data Simple Program

Using Pinata SDK to upload the JSON Data to IPFS .


## .env Reference

#### .env example

| Parameter | Type     | URL                |
| :-------- | :------- | :------------------------- |
| PINATA_SDK_API_KEY | `string` | https://www.pinata.cloud/ |
| PINATA_SDK_API_SECRET | `string` | https://www.pinata.cloud/ |

#### Install the package

```
  npm i
```
#### Start the program

```
  npm start
```
